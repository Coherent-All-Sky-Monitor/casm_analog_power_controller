API Usage Examples

Turn ON Stack 0, Relay 1:

curl -X POST http://localhost:5001/api/relay/0/1 -H "Content-Type: application/json" -d '{"state": 1}'

Turn OFF Stack 0, Relay 1:

curl -X POST http://localhost:5001/api/relay/0/1 -H "Content-Type: application/json" -d '{"state": 0}'

Get Stack 0, Relay 1 state:

curl http://localhost:5001/api/relay/0/1

Get all relays in Stack 0:

curl http://localhost:5001/api/relay/stack/0

Get all relay states:

curl http://localhost:5001/api/relay/all
